# 无向图的邻接矩阵与邻接表表示

## 题目

给定 6 个顶点 (A, B, C, D, E, F) 的无向图，分别写出其**邻接矩阵**和**邻接表**表示。

## 图的边关系

根据图示，各顶点的邻接关系如下：

| 顶点 | 邻接顶点 | 度数 |
|:---:|:---:|:---:|
| A | B, C, F | 3 |
| B | A, C, E | 3 |
| C | A, B, D | 3 |
| D | C, E | 2 |
| E | B, D, F | 3 |
| F | A, E | 2 |

**总边数**: 8 条

---

## 一、邻接矩阵 (Adjacency Matrix)

邻接矩阵是一个 $n \times n$ 的方阵（$n$ 为顶点数），其中：
- $matrix[i][j] = 1$ 表示顶点 $i$ 和顶点 $j$ 之间有边相连
- $matrix[i][j] = 0$ 表示没有边

对于**无向图**，邻接矩阵是**对称矩阵**（$matrix[i][j] = matrix[j][i]$）。

### 矩阵表示

|  | **A** | **B** | **C** | **D** | **E** | **F** |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| **A** | 0 | 1 | 1 | 0 | 0 | 1 |
| **B** | 1 | 0 | 1 | 0 | 1 | 0 |
| **C** | 1 | 1 | 0 | 1 | 0 | 0 |
| **D** | 0 | 0 | 1 | 0 | 1 | 0 |
| **E** | 0 | 1 | 0 | 1 | 0 | 1 |
| **F** | 1 | 0 | 0 | 0 | 1 | 0 |

### 矩阵特点

- **对称性**: 矩阵关于主对角线对称（无向图特性）
- **主对角线**: 全为 0（无自环）
- **1 的个数**: 每行 1 的个数 = 该顶点的度数
- **总 1 的个数**: 16 个（每条边被计算两次，$2 \times 8 = 16$）

---

## 二、邻接表 (Adjacency List)

邻接表为每个顶点维护一个链表（或数组），存储其所有邻接顶点。

### 邻接表表示

```
A: [B] -> [C] -> [F] -> NULL
B: [A] -> [C] -> [E] -> NULL
C: [A] -> [B] -> [D] -> NULL
D: [C] -> [E] -> NULL
E: [B] -> [D] -> [F] -> NULL
F: [A] -> [E] -> NULL
```

### 简化表示

| 顶点 | 邻接表 |
|:---:|:---|
| A | B &rarr; C &rarr; F |
| B | A &rarr; C &rarr; E |
| C | A &rarr; B &rarr; D |
| D | C &rarr; E |
| E | B &rarr; D &rarr; F |
| F | A &rarr; E |

---

## 三、两种表示法的对比

| 特性 | 邻接矩阵 | 邻接表 |
|:---:|:---:|:---:|
| **空间复杂度** | $O(n^2)$ | $O(n + m)$ |
| **判断边是否存在** | $O(1)$ | $O(deg(v))$ |
| **遍历某顶点的邻接点** | $O(n)$ | $O(deg(v))$ |
| **适合场景** | 稠密图（边数接近 $n^2$） | 稀疏图（边数远小于 $n^2$） |
| **本题适用性** | 可用（$n=6$ 较小） | **更优**（$m=8$ 稀疏） |

> 本题中 $n=6, m=8$，属于稀疏图，**邻接表更节省空间**。

---

## 四、验证

### 对称性验证

邻接矩阵满足 $matrix[i][j] = matrix[j][i]$，符合无向图特性。

### 度数验证

$$\sum_{v \in V} deg(v) = 3 + 3 + 3 + 2 + 3 + 2 = 16 = 2m$$

满足**握手定理**：所有顶点度数之和 = $2 \times$ 边数。

```python
    [0, 0, 1, 0, 1, 0],  # D
    [0, 1, 0, 1, 0, 1],  # E
    [1, 0, 0, 0, 1, 0],  # F
]
vertices = ['A', 'B', 'C', 'D', 'E', 'F']

# 邻接表表示
adj_list = {
    'A': ['B', 'C', 'F'],
    'B': ['A', 'C', 'E'],
    'C': ['A', 'B', 'D'],
    'D': ['C', 'E'],
    'E': ['B', 'D', 'F'],
    'F': ['A', 'E']
}
```
